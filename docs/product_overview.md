# Product Overview

## 项目定义

`zhumeng-qiangji-skill` 是一个面向强基计划场景的 AI skill repository。
它的职责不是直接替代政策数据库，而是作为模型推理时的：

- 领域策略层
- 风险控制层
- 路由与分流层
- 结构化输出协议层

## 核心用户问题

该仓库主要服务于以下问题类型：

- 政策梳理：某校某年是否招生、模式如何、考核节奏怎样
- 条件判断：某省、某选科、某背景能否报名
- 分数解释：某个分数、分位、综合分到底是什么意思
- 择校建议：在信息充分前提下做初筛与保守分层
- 备考建议：是否适合强基、偏笔试还是偏面试、先做什么
- 服务分流：用户应该进入政策梳理、择校规划还是校测训练

## 关键价值

### 对大模型

- 降低强基计划场景下的幻觉率
- 把高风险推断显式降级
- 把问题先结构化再回答

### 对产品

- 为后续接入知识库、检索系统、咨询系统提供稳定接口
- 把 prompt、rule、schema、eval 放进统一版本库

### 对运营与教研

- 可以逐步沉淀术语、规则、案例、失败模式和评测集

## 非目标

本仓库当前不负责：

- 建立完整政策数据库
- 提供招生官网级别的实时官方更新
- 实现前端问答产品
- 接入真实外部 API 或在线检索
