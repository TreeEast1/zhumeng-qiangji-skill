# Terminology

## 必须严格区分的术语

- `高考裸分`
  - 指高考文化课原始分数，不等于综合成绩
- `入围线`
  - 指进入校测或后续环节的门槛，不等于最终录取线
- `录取线`
  - 指最终录取所对应的边界结果，不等于入围线
- `综合成绩`
  - 指按高校规则合成后的成绩，通常包含高考成绩与校测表现
- `校测成绩`
  - 指笔试、面试、体测等校测环节成绩
- `位次`
  - 指考生在省内排序，不等于高考裸分
- `位次对应分`
  - 为便于估算而使用的分数映射，不等于真实录取结果
- `出分前校测`
  - 高考成绩公布前进行校测
- `出分后校测`
  - 高考成绩公布后进行校测

## 术语约束

任何输出若出现具体数值，必须尽量同时说明：

- 该数值属于哪种分数类型
- 所绑定的年份
- 所绑定的省份
